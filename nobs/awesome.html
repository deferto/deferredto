Awesome DeferredTo

DeferredTo makes page loading faster by deferring image, media, plug-in and frame loading to the moment they are just about to appear on the screen. DeferredTo does not break page layout because it loads the image width and height info while the page is loading. So, the page elements has the same layout and box size as if they were fully loaded from Internet.
